This folder contain assignment 4 which is Multilayer Perceptron need to be implemented using different activation function.

Dataset - I used iris data set consists of 50 samples from each of 3 species of Iris(Iris setosa, Iris viginica and Iris Veriscolor) having 4 features.

MLP Architecture - input layer with 4 input features and one bias.
		   one hidden layer with 5 node and one bias.
		   output layer with 3 node output.

ABOUT .m FILE:
1) iris.m , sigmoid.m - mlp using sigmoid activation function.
2) iris_htan.m, hyp_tan.m - mlp using hyperbolic tangent function.
3) iris_relu.m, deri_relu.m - mlp using rectifier linear units.
4) iris_leakyrelu.m, deri_l_relu.m - mlp using leaky reactifer linear units.
5) iris_elu.m, elu.m, deri_elu.m - mlp using exponential linear units.
